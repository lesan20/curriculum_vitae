Бонитировка скота - оценка списка параметров животного (например, ширина грудной клетки) специалистом вручную, это стрессовая процедура для животного и времязатратная для предприятия. Заказчик хотел автоматизировать бонитировку с помощью компьютерного зрения.

  - Изучила научные статьи по заданной теме, собрала релевантные данные и адаптировала их в соответствии с требуемым списком параметров бонитировки
  - На основе списка параметров определила минимально необходимое и достаточное расположение камер, подобрала характеристики для оценки каждого из параметров
  - Разработала макет конструкции согласно потребностям CV, подобрала оборудование, совместно с конструкторами заказчика внесли корректировки
  - Подготовила техническое описание системы и предоставляла консультации при оформлении патента 
  - Разработала макет ПО для специалиста, который будет собирать конструкцию и контролировать работу системы
  - Реализация макета ПО для сбора датасета, который давал инструкции по подключению системы и контролировал ее готовность к запуску: проверка подключения и калибровки всех камер, тестовая запись и оценка корректности сохранения данных. Сбор данных проведен успешно. Стек: Flask, Docker.
  - Разработан алгоритм детектирования животных и выделения значимых областей.

### Разработка прототипа алгоритма односторонней птихографии
Работа в команде  LiberTEM, научная лаборатория в Германии. [Информация о проекте LiberTEM](https://libertem.github.io/LiberTEM/applications.html)  
Разработка высокопроизводительной реализации птихографии с односторонней полосой (single side band ptychography):
- Анализ эталонной реализации для офлайн-данных
- Поиск аналога для математически эквивалентного алгоритма, позволяющего объединять частичные результаты обработки неполных данных, следуя стратегии MapReduce
- Создание вычислительно интенсивного алгоритма для обработки больших данных электронной микроскопии, подходящего для работы в реальном времени на распределённых системах
- Разработка высокопроизводительной реализации альтернативного алгоритма в рамках фреймворка LiberTEM. Стек: Python, NumPy, Pytest и Numba.
- Результат: Время работы алгоритма сокращено с нескольких часов до реального времени; изображение формируется во время сканирования микроскопа.
  
[Научная статья]([https://arxiv.org/abs/2106.13457](https://academic.oup.com/mam/article/27/5/1078/6888074?login=false)) "Live processing of momentum-resolved STEM data for first moment imaging and ptychographyLive processing of momentum-resolved STEM data for first moment imaging and ptychography", опубликована на первой странице Microscopy and Microanalysis

  - Рассчитаны диапазоны глубины зондирования в соответствии с анатомическими особенностями миокарда, предложена методика зондирования и калибровки зонда
  - Предложены варианты конструкции зонда
  - Автоматизирована обработка экспериментальных данных: разработано программное обеспечение, включающее удобный интерфейс и базу данных для хранения экспериментальных данных. Функционал позволяет загружать новый файл данных, рассчитывать векторы признаков, выводить графики и сохранять обработанные данные в базе данных. Стек: MATLAB
